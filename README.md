#月度绩效测评
 
> 印管部月度绩效测评。Vue+VueX+Vue-Router+Vue-Resource + Webpack开发。

## 编译步骤

``` bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 编译生成
npm run build
```
